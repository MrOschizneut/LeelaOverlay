

This is a manifest V2 extension.

# Usage
1. Download it
2. Download [lc0](https://lczero.org/) binaries and place them on a subfolder "lc0/" inside
3. Run server.py
4. Load the extension/manifest.json via your browser's tools
5. Start a lichess game
